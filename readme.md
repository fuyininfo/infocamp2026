## 关于光谷科学岛的艺术黑客松，总时长10天限定学员10人。
在这10天里，我们会提供食物和住宿，及基础的创作工具与网络支持。复印info会和大家一起，将其过去积累的DIY技术、开源理念和协作经验，带入到光谷这片象征着科技与资本交织的科学岛语境中。白天，我们将围绕科学岛及其周边空间展开田野调查、讨论技术开发与艺术干预的可能；夜晚，大家则聚在一起分享各自的档案、代码或文本，经历头脑风暴。我们期待在这10天结束时，能共同产出一系列具体的方案——或许是一个开源工具、一本合作的zine、一件基于本地材料的装置，或是一次出其不意的空间介入。我们邀请艺术家、写作者、程序员和在地行动者们一起，在共居、共创中，用低保真的手段和人文视角的黑客精神，去探讨技术基础设施与日常生存、个体实践之间的张力，寻找重塑公共空间的可能方案。

>意向时间：4月初或五月初
>共同学员：子杰、jojo等
