## 到科学岛去！（在科学岛上）

# [cite_start]最近我们打算在新一年里做infocamp研习营：关于光谷科学岛的艺术黑客松，总时长10天限定学员10人。 [cite: 7]
[cite_start]项目地址：[https://github.com/fuyininfo/infocamp2026](https://github.com/fuyininfo/infocamp2026) [cite: 7]

***

[cite_start]在这10天里，我们会提供食物和住宿，及基础的创作工具与网络支持。 [cite: 7] [cite_start]复印info会和大家一起，将其过去积累的DIY技术、开源理念和协作经验，带入到光谷这片象征着科技与资本交织的科学岛语境中。 [cite: 7] [cite_start]白天，我们将围绕科学岛及其周边空间展开田野调查、讨论技术开发与艺术干预的可能；夜晚，大家则聚在一起分享各自的档案、代码或文本，经历头脑风暴。 [cite: 7] [cite_start]我们期待在这10天结束时，能共同产出一系列具体的方案——或许是一个开源工具、一本合作的zine、一件基于本地材料的装置，或是一次出其不意的空间介入。 [cite: 7] [cite_start]我们邀请艺术家、写作者、程序员和在地行动者们一起，在共居、共创中，用低保真的手段和人文视角的黑客精神，去探讨技术基础设施与日常生存、个体实践之间的张力，寻找重塑公共空间的可能方案。 [cite: 7]

> [cite_start]* **主题：** 科学岛 [cite: 7]
> [cite_start]* **关键词：** 技术空间，空间介入，媒介考古，人机关系，写作，创造性工作，多重未来，多元技术-时间，电子垃圾，开源，硬件 [cite: 7]
> [cite_start]* **意向时间：** 4月初或5月初 [cite: 7]
> [cite_start]* **共同学员：** 子杰、jojo等 [cite: 7]
> [cite_start]* **可供使用的工具：** Brother编织机（AYAB主板改造版），Stencil版画工具箱、各种元件，各种硬件如pc-1500、合成器、树莓派、esp32、烤箱等；以及用于基础网络与计算机硬件改造的工具。你也可以自己带工具来。 [cite: 7]
> [cite_start]* **会提及和使用的技术：** openclaw和硬件互动 [cite: 7]
> [cite_start]* **讲座和讨论：** （待定，正在陆续联系；希望可以分享关于媒介考古、社会参与式艺术和空间介入、身体工作坊，电子音乐和合成器，电子垃圾等方面的内容。） [cite: 7]
> [cite_start]* **费用：** 1200元（每人每日住宿60元覆盖房租水电成本+食物30元+30元劳动报酬，总共10天） [cite: 7]
